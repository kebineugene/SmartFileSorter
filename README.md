# 📂 SmartFileSorter v1.0 (Windows)

Удобная и современная графическая программа, которая автоматически наводит порядок в замусоренных папках на вашем компьютере. Она мгновенно сканирует выбранное место, определяет типы файлов и аккуратно раскладывает их по полочкам (Изображения, Документы, Музыка, Видео, Архивы, Образы дисков и Виртуализация).

## ✨ Что умеет SmartFileSorter:
* **Уборка в один клик:** Просто укажите любую замусоренную папку через привычный проводник Windows.
* **Автоматический подсчет веса:** Программа сама рассчитает и покажет вам общий размер файлов до начала автоматической уборки.
* **Наглядный прогресс-бар:** Плавная зеленая полоса выполнения покажет статус сортировки файлов в реальном времени.
* **Умная защита (Дисклеймер):** Встроенное интерактивное предупреждение защитит вас от случайного изменения путей в важных папках (например, в загрузках Telegram), чтобы другие программы не потеряли доступ к вашим файлам.
* **Адаптивный дизайн:** Интерфейс автоматически подстраивается под темную или светлую тему вашей операционной системы Windows и отлично читается на любых мониторах (включая 2K и 4K).

## 📁 Структура проекта
* `organizer.py` — чистая логика (backend), сканирование и безопасное перемещение файлов.
* `main.py` — графическая оболочка (frontend) и обработка событий пользователя.

## 🚀 Инструкция по запуску

### Вариант 1. Для обычных пользователей (Быстрый запуск)
1. Перейдите в раздел **Releases** (в правой колонке этой страницы) и скачайте актуальный файл **`SmartFileSorter.exe`**.
2. Перенесите скачанный файл в любое удобное место на жестком диске.
3. Запустите программу двойным кликом. Установка Python или дополнительных библиотек не требуется!

### Вариант 2. Для разработчиков (Запуск из исходного кода)
Если вы хотите изучить проект или запустить его напрямую через исходный код Python:
1. Установите необходимую графическую библиотеку `customtkinter`:
   ```bash
   pip install customtkinter
   ```
2. Запустите главный файл интерфейса:
   ```bash
   python main.py
   ```

## ⚠️ Важное предупреждение:
Не используйте программу для сортировки сетевых папок (NAS-хранилищ или общих дисков в локальной сети). Это может сильно замедлить работу сети или привести к сетевым ошибкам перемещения файлов.

## 🛠️ Стек технологий
* **Language:** Python 3.x
* **GUI Framework:** `CustomTkinter` (Tkinter под капотом)
* **File Management:** `pathlib`, `shutil`, `os`
